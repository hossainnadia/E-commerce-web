# 🛒 E-commerce-web

একটি আধুনিক ই-কমার্স ওয়েবসাইট, যা React এবং Tailwind CSS ব্যবহার করে তৈরি করা হয়েছে।

## 🔗 প্রজেক্ট লিঙ্ক (Project Links)

- **Live Website:** https://e-commerce-web-snowy-psi.vercel.app
- **GitHub Repository:** https://github.com/hossainnadia/E-commerce-web.git

---

## ✨ প্রজেক্টের মূল ফিচারসমূহ (Features)

- **ডাইনামিক লেআউট:** ইউজার লগইন অবস্থায় আছে কি নেই, তার ওপর ভিত্তি করে নেববার এবং ফুটার অটোমেটিক হাইড বা শো হয়।
- **প্রোটেক্টড রাউট (Protected Routes):** কোনো ইউজার লগইন না করে সরাসরি কার্ট (`/cart`) বা ড্যাশবোর্ড পেজে ঢুকতে চাইলে তাকে লগইন পেজে রিডাইরেক্ট করে দেওয়া হয়।
- **সরাসরি কিউআর কোড:** ফুটারে মার্চেন্ট কিউআর কোডের আসল ইমেজ যুক্ত করা আছে, যা সরাসরি স্ক্যান করা সম্ভব।
- **রেসপন্সিভ ডিজাইন:** মোবাইল, ট্যাবলেট এবং কম্পিউটার—সব ডিভাইসের স্ক্রিনের সাথে ওয়েবসাইটটি পারফেক্টলি মানিয়ে নেয়।

---

## 🚀 টেক স্ট্যাক (Tech Stack)

- **Frontend:** React.js, React Router DOM, Tailwind CSS
- **Icons:** Lucide React
